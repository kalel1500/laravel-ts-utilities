## Instalación

```bash
npm laravel-ts-utilities
```


## Comandos para el desarrollo

- `npm deprecate laravel-ts-utilities@*** "Versión ..., no usar"`: deprecar una version publicada que no sirve
- `npm version patch`
- `npm version minor`
- `npm version major`
- `npm publish`
